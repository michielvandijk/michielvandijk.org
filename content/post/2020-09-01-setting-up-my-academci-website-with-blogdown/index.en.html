I finally found some time during the holidays to setup my personal academic website using the fantastic R package blogdown. Several others have written excellent blogs and documentation on how to get started with blogdown so I will mention the sources I used below and not repeat the basics. Instead I will focus on a few issues that come right after creating the template website and took me quite some time to figure out (although in many cases it was just a matter of carefully reading Hugo documentation).

My main guide to setting up the first version was the blog by [ADD]. I mainly decided to use her guidance as it was one of the most recent blogs on setting up blogdown and therefore was most likely to cover possible updates in Hugo or the Hugo Academic theme that might have been implemented and not (yet) covered by the otherwise excellent book by Yihie et al [ADD], which contains most information on how to setup and use blogdown.
